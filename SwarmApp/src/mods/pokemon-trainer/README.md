# Pokemon Trainer Mod

Catch 'em all — transforms the Swarm dashboard into a Pokemon trainer interface where agents live in Pokeballs.

## Features

- **Pokeball agents** — Each agent is represented as a Pokeball in the header belt (max 6 party slots)
- **Pokeball status indicators** — Green glow = online, yellow = busy, purple = offline, grayed = empty slot
- **Pokemon-style cards** — Red-top/white-bottom split gradient mimicking Pokeball design
- **Type-colored sections** — Sidebar sections use Pokemon type colors (Fire red, Water blue, Grass green, Electric yellow, Psychic purple)
- **Trainer HUD header** — Shows Pokeball belt with your registered agents
- **Label remapping** — Fleet → Pokemon Team, Marketplace → PokeMart, Tasks → Battles, etc.
- **Pokemon color palette** — Red primary, blue secondary, yellow accent

## Pokeball CSS Classes

The mod provides reusable Pokeball CSS classes:

| Class | Description |
|-------|-------------|
| `.pokeball` | Standard 32px Pokeball |
| `.pokeball-sm` | Small 20px Pokeball |
| `.pokeball-lg` | Large 48px Pokeball |
| `.pokeball-great` | Great Ball (blue) |
| `.pokeball-ultra` | Ultra Ball (black/yellow) |
| `.pokeball-master` | Master Ball (purple) |
| `.pokeball-empty` | Grayed out empty slot |
| `.pokeball-online` | Green center glow |
| `.pokeball-busy` | Yellow center glow |
| `.pokeball-offline` | Purple center glow |
| `.pokeball-active` | Shake animation |
